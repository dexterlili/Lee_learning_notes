代码规范：
- 定义函数
```
- fn main(){}
- 
- println!是一个Rust macro宏，如果是函数就没有"!"
```
- 编译相关
```
rust的缩进是4个空格而不是tab
- rustc 源文件名  //编译命令，只适合简单的rust程序
Cargo
- cargo new 新项目名  //创建新项目
	- Carge.html  //配置文件
	- package  //区域标题
	- dependencies  //依赖项
	- crate //第三方库名
- cargo build  //创建可执行文件，编译（一般用于调试使用）
- cargo run  //编译+执行
- cargo check  //检查代码，确保编译可以通过，不产生可执行文件
	- 比cargo build快得多
- cargo build --release //给正式版编译一个可执行文件
```
